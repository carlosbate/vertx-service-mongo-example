## vertx-service-mongo-example

Shows an example on how to use Vert.x service proxy with MongoDB and a HTTP Server.

This project uses the following dependencies:
  - vertx-core
  - vertx-web
  - vertx-codegen
  - vertx-service-proxy
  - vertx-service-factory
  - vertx-mongo-service
